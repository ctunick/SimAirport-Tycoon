# Contributing to SimAirport Tycoon

First off, thank you for considering contributing! We welcome contributors of all experience levels.

## How to Contribute

1.  **Find an Issue:** Look for open issues on our GitHub repository. Good first issues are labeled `good first issue`.
2.  **Fork the Repo:** Fork the project to your own GitHub account.
3.  **Create a Branch:** Create a new branch for your feature or bugfix.
4.  **Develop:** Make your changes. Please follow the guidelines in `docs/Development-Guidelines.md`.
5.  **Test:** Ensure your changes work as expected and don't break existing functionality.
6.  **Submit a Pull Request:** Open a pull request from your branch to the main repository. Provide a clear description of your changes.

## Code Style

- **C# (Unity):** Follow the conventions in `docs/Development-Guidelines.md`.
- **Python:** Follow PEP 8 standards.

We look forward to your contributions!
