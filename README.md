## Vision

SimAirport Tycoon aims to capture the joy of designing and optimizing a living airport:

* Place terminals, check-in, security, gates, and services on a grid.
* Watch AI passengers flow through your designs.
* Balance costs and revenue.
* Unlock new buildings and mechanics as your airport grows.

Longer-term, the vision includes:

* **Co-op / multiplayer support (post-1.0)**
  Multiple players collaboratively designing and managing a shared airport via a command-based simulation core.

  > Co-op is a **long-term goal** and will **not** be attempted before at least version **1.0.0**, after the single-player experience is stable and feature-complete.

* **Additional platforms**

  * PC (primary target for development)
  * Eventually **iOS** and other platforms, once the core simulation is stable and performant.

* **Deep simulation & modding**
  Data-driven configs, Python tools for generating schedules and balancing, and a clean architecture that invites experimentation.

## Current Scope (v0.0.1)

The initial vertical slice focuses on:

* **Basic world building**

  * Grid-based placement of core facilities (entrance, check-in, security, gates, restrooms).

## Getting Started

### 1. Prerequisites

Youâ€™ll need:

* **Git**
* **Unity (LTS 3D / URP)**

  * Install via Unity Hub.
* **Python 3.x**

  * Recommended: 3.10+ for tooling.

### 2. Clone the Repository

```bash
git clone https://github.com/<your-org-or-user>/simairport-tycoon.git
cd simairport-tycoon
```

### 3. Generate Data (Optional for early builds)

From the `python/` directory:

```bash
cd python
python -m venv .venv
source .venv/bin/activate    # or .venv\Scripts\activate on Windows
pip install -r requirements.txt  # if present

python tools/export_to_json.py   # generates basic flight schedules & configs
```

This will create JSON config files under:

```text
unity/AirportTycoonUnity/Assets/Data/
```

### 4. Open in Unity

1. Open **Unity Hub**.
2. Click **Open** and select `unity/AirportTycoonUnity/`.
3. Open the `DemoScene` (usually under `Assets/Scenes/`).
4. Press **Play** to run the current vertical slice.

You should see:

* A simple airport layout.
* AI passengers moving through core zones.
* Money changing as passengers successfully board.
