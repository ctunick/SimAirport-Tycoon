\# Python Tools for SimAirport Tycoon



This directory contains helper scripts for generating game data (flight schedules, passengers, etc.) and exporting it into the Unity project.



\## Python Version



\- \*\*Required Python version:\*\* 3.10+ (3.11 recommended)



\## Setup



From this `python/` folder:



```bash

python -m venv .venv

\# Windows

.venv\\Scripts\\activate

\# macOS / Linux

source .venv/bin/activate



\# Install dependencies

pip install pyyaml
